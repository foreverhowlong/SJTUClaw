# 课程报告模板

按课程要求选择和改写章节；删除所有提示文字后再交付。没有可靠来源时，不要为了填满“参考材料”而虚构条目。

## 论文、读书报告或学习总结

```markdown
# 能体现核心问题或结论的标题

> 课程：
> 姓名 / 学号：（仅在用户要求且已提供时填写）

## 摘要

用一段话说明研究或阅读对象、核心问题、采用的材料或分析路径，以及主要结论。

**关键词：** 3–5 个

## 一、问题背景与核心问题

交代必要背景、课程语境和本文要回答的问题，避免大段空泛铺垫。

## 二、材料、概念与分析依据

说明实际使用的材料和关键概念，为后续论证建立边界。

## 三、分析与讨论

围绕中心论点组织若干小节。每节解决一个子问题，并说明证据如何支持结论。

## 四、结论与反思

直接回应核心问题，概括主要发现，并指出局限、课程收获或后续问题。

## 参考材料

仅列出实际读取或由用户明确提供的信息；格式服从课程要求。
```

## 实验报告

```markdown
# 实验名称

## 一、实验目标与原理

## 二、实验条件、材料与方法

## 三、结果

客观呈现实测数据、现象、图表或运行结果，不提前混入原因解释。

## 四、分析与讨论

解释结果、对照预期，分析误差来源、异常现象和方法限制。

## 五、结论

说明实验是否达到目标，以及结果能够支持什么结论。

## 参考材料或附录
```
